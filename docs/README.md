## 项目简介

TrulyMEM是一个让AI拥有长期记忆能力的图记忆系统，通过图数据库存储实体关系，让AI能够像人类一样记忆、回忆和管理信息。

## 核心特性

- 长期记忆存储
- 人设图机制
- 流式消息显示
- 跨平台支持
- 独立部署
