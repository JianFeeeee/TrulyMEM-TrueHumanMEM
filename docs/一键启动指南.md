# 一键启动指南

## 🚀 快速开始

### Windows

```bash
# 方法1: 双击运行
双击 start.bat

# 方法2: 命令行
start.bat

# 方法3: Python
python start.py
```

### Linux / macOS

```bash
# 方法1: Shell脚本
chmod +x start.sh
./start.sh

# 方法2: Python
python3 start.py
```

## 📋 启动流程

```
[Step 1/3] 检查 Python
    ↓
[Step 2/3] 设置虚拟环境
    - 创建venv（如果不存在）
    - 安装依赖
    ↓
[Step 3/3] 启动应用
    ↓
✅ 系统初始化成功！
```

## ⏱️ 启动时间

- **首次启动**: 约1-2分钟（安装依赖）
- **后续启动**: 约30秒

## 📝 首次使用

### 1. 启动应用

```bash
start.bat  # Windows
./start.sh # Linux/macOS
```

### 2. 配置 API Key

1. 按 **F2** 展开侧边栏
2. 点击 **"配置"**
3. 输入 **DeepSeek API Key**
4. 按 **Enter** 保存

### 3. 开始对话

- 输入消息
- 按 **Enter** 发送

## ⚠️ 常见问题

### Q: Python 未找到

**A:** 安装 Python 3.8+
- Windows: https://www.python.org/downloads/
- Linux: `sudo apt install python3`
- macOS: `brew install python3`

### Q: 依赖安装失败

**A:** 手动安装
```bash
python -m venv venv
venv\Scripts\activate  # Windows
source venv/bin/activate  # Linux/macOS
pip install -r requirements.txt
```

### Q: API Key 无效

**A:** 检查API Key格式
- 应以 `sk-` 开头
- 无多余空格
- 正确复制
